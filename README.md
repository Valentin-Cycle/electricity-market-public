# electricity-market-public

`market.json` — données publiques de l'**Observatoire des marchés de l'électricité**
(page publique cycle.energy). Fichier régénéré automatiquement par le pipeline du
dashboard (repo privé). Aucun secret.

- Prix spot day-ahead du jour (ENTSO-E)
- Prix de capacité réserves FCR/aFRR (RTE), normalisés en €/MW/h
- Profil horaire normalisé par année
- Cumul annuel des heures à prix négatif
